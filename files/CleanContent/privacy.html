Privacy Policy

CleanContent does not store or share any user data obtained from Meta platforms. We use the Instagram Graph API to access comment data only for the authorized account and only to enable moderation actions directly within the app.

All data is processed securely and temporarily. We do not store or analyze personal data beyond the immediate session needed to display and manage comments.

Users may revoke access at any time via their Instagram account settings.

Data Usage

We only request permissions necessary to provide moderation functionality, such as instagram_business_basic and instagram_business_manage_comments. These are used to fetch business account info and manage comments, respectively.

Security

All interactions with Meta APIs are encrypted and authenticated. No data is shared with third parties.
